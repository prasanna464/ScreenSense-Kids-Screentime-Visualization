# ScreenSense: Kids' Screentime Visualization

## Project Overview
This project analyzes kids' screen time patterns to uncover trends based on age, gender, device type, and location using data visualization.

## Internship
This project is developed as part of the **Infosys Springboard Virtual Internship 6.0**.

## Dataset
Indian Kids Screentime 2025  
https://www.kaggle.com/datasets/ankushpanday2/indian-kids-screentime-2025

## Tech Stack
Python  
NumPy  
Pandas  
Matplotlib  
Seaborn  
Power BI
